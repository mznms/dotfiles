# mznms' dotfiles

1. Install [chezmoi](https://www.chezmoi.io/)

   - Use your favorite package manager or just run the following:

     - ```sh
        sh -c "$(curl -fsLS get.chezmoi.io)"
       ```

2. Download this repo and apply the dotfiles.

   - ```sh
      chezmoi init --apply mznms
     ```
